# The Blooming Vase

A digital flower arrangement focused on simple, interactive movement.

## About
This project features a minimalist vase where the flowers are the main focus. By clicking on each flower, you can toggle its state to watch it open or close. It is a study in clean design and smooth transitions.

## Live Demo
You can view the project here:
[View the Interactive Flowers](https://joybytez.github.io/click-to-bloom/)

## Interaction
- Click any flower to toggle its bloom.
- Each interaction is handled through vanilla JavaScript and CSS transitions.
